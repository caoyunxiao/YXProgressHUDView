Welcome to YXProgressHUDView!
===================

1. 如何使用？

使用cocoapods 导入即可

pod 'YXProgressHUDView', '~> 1.0.2'
HUD_Configuration.h文件可自己配置
自定义图片需替换下Resources目录下的图片。

>  pod 'YXProgressHUDView'
